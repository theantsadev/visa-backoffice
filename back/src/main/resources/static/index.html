Visa Backoffice

Accueil des fonctionnalites

Cette page centralise les routes principales du projet : parcours utilisateur, saisie metier et soumission finale.

Commencer une nouvelle demande Consulter les demandes Voir la page de confirmation

Pages UI

Parcours principal

GET
/demande/nouveau
Formulaire multi-etapes avec validations et appels API dans l ordre metier.

Liste et detail des demandes

GET
/demande/liste
Vue backoffice pour lister les demandes et consulter le detail complet d'un dossier.

Confirmation

GET
/demande/confirmation
Affiche le resultat de la soumission : id, statut, dateDemande.

Routes API - Saisie de demande

Creation demandeur

POST
/api/demandeurs
Reçoit DemandeurDTO et retourne le demandeur cree avec son id.

Creation passeport

POST
/api/demandeurs/{idDemandeur}/passports
Reçoit PassportDTO et retourne le passeport cree.

Creation visa transformable

POST
/api/passports/{idPassport}/visas-transformables
Reçoit VisaTransformableDTO et retourne le visa cree.

Soumission finale

POST
/api/demandes/nouveau-titre
Reçoit DemandeDTO et retourne id, statut, dateDemande.
